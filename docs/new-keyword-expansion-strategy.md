## 二、横向拓展方向

FocusCoach 当前核心：ADHD adults → focus training。

横向拓展可以把同样的人群和场景扩大 3-5 倍：

### 方向 1：Neurodivergent 人群

- 关键词：neurodivergent productivity, autism ADHD focus tools, AuDHD focus strategies
- 内容：不修改产品定位，只扩展内容覆盖
- 示例文章：
  - "Focus Tools for Neurodivergent Adults: A Practical Guide"
  - "AuDHD and Focus: What Works When Your Brain is Both"

### 方向 2：Remote Work / 知识工作者

- 关键词：remote work focus, work from home ADHD, deep work for distracted brains
- 内容：把 ADHD 工具包装成「任何知识工作者都可用」
- 示例文章：
  - "How to Focus Working From Home: A 10-Minute Routine"
  - "ADHD Productivity Hacks for Remote Teams"

### 方向 3：Executive Function

- 关键词：executive function exercises, executive dysfunction tools, working memory training
- 内容：更偏「认知能力训练」，不只是 ADHD
- 示例文章：
  - "Executive Function Exercises You Can Do in 10 Minutes"
  - "Working Memory Training for Adults: What Actually Helps"

### 方向 4：Mental Fitness / Wellness

- 关键词：mental fitness app, cognitive wellness, brain health adults
- 内容：把训练包装成「大脑健身」
- 示例文章：
  - "Mental Fitness for Adults: A Beginner's Routine"
  - "Brain Training vs Meditation: Which Should You Do First?"

## 三、高权重平台内容发布计划

每篇新词文章按以下流程发布：

1. **主站首发**：`/blog/{new-keyword-slug}`
2. **Medium republication**：发布摘要 + 全文链接，canonical 指向主站
3. **Dev.to**：发技术/产品视角文章，底部带主站链接
4. **LinkedIn**：把文章核心观点拆成 3-5 段 post
5. **Twitter/X**：拆成 thread，@ 相关账号
6. **Reddit**：在对应 subreddit 发讨论帖（不是硬广）

### 平台账号要求

- Medium：需要账号
- Dev.to：需要账号 + API Key
- LinkedIn：个人账号即可
- Twitter/X：需要账号
- Reddit：需要账号 + karma

## 五、新词执行流程

```
Step 1: 从竞品监控/谷歌趋势/Reddit讨论中发现新词
Step 2: 用 Google Trends + Ahrefs 免费版验证搜索量
Step 3: 在 /blog 发布 1200-1800 字文章（3-5 天内完成）
Step 4: 同步到 Medium + Dev.to + LinkedIn + Twitter
Step 5: 在 Reddit 相关 subreddit 发讨论帖
Step 6: 提交到 3-5 个目录/资源页
Step 7: 2 周后检查排名，如果进入前 30，追加外链巩固
```

## 八、关键提醒

1. **新词要快**：发现一个有效新词，3 天内必须发布页面。
2. **不要只追量**：优先追「有搜索量 + 低竞争 + 与产品相关」的词。
3. **外链要相关**：ADHD/心理健康/生产力相关的外链优先，不相关的外链宁可不要。
4. **Canonical 必须正确**：Medium/Dev.to  republication 必须 canonical 回主站。
5. **不要医疗化**：所有内容保持「生产力工具，非医疗设备」定位。
