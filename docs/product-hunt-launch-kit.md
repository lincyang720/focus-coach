**Short description:** FocusCoach helps adults transition into focused work with a free 60-second attention check, five adaptive exercises, and plain-language weekly patterns. It is a productivity tool—not a medical device, diagnosis, or ADHD treatment.

**Topics:** Productivity, Web App, Artificial Intelligence, ADHD, Quantified Self

## Maker comment

Hi Product Hunt — I built FocusCoach because most focus tools start with a timer, while the hardest part for many of us is making the transition into the task.

FocusCoach starts with a no-signup, 60-second attention check. The full app includes Number Memory, Quick Match, N-Back, Task Switch, and Stroop exercises, with adaptive difficulty and weekly AI summaries. The intended loop is short: practice, review, choose the next action, and return to work.

FocusCoach is deliberately positioned as a productivity and attention-training tool. It does not diagnose, treat, or cure ADHD, and it does not promise that a game score transfers to everyday performance.

I would especially value feedback on three things:

1. Does the homepage explain the product quickly?
2. Does the 60-second check make it easier to begin a work block?
3. Which result or recommendation would make you return tomorrow?

Thanks for trying it.

## Link-exchange rule

Do not offer reciprocal links as a condition of placement. Exchange links only when each resource independently helps the other page’s readers. Avoid sitewide footer links, exact-match anchor quotas, paid link schemes, and unrelated directories. Record the referring page, destination page, anchor, relationship, and date so the pattern stays natural.
